# 🎲 UnluckyCraft
**Unlucky Minecraft is coming!**  
A Minecraft plugin that adds random chance-based rewards!

---

## ⚙️ Config

```yaml
# - - - - - - - - - - #
#     UnluckyCraft    #
#                     #
#   by SuperCHIROK1   #
# - - - - - - - - - - #

chances:
  diamond: 0.25
  emerald: 0.2
  gold: 0.15
  iron: 0.2
  coal: 0.3
  quartz: 0.3
  netherite: 0.05
```

## 📦 Commands

/ulc <about|reload> – About info or reload the plugin

/uls – Open the chance-setting GUI
